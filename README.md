# 🛍️ AI Smart Store Assistant

An intelligent e-commerce web application that helps users discover products using AI-powered recommendations. Built as part of a cloud deployment project and deployed using Google Cloud Run.

---

## 🚀 Live Demo

👉 (https://ai-smart-store-423447867140.us-central1.run.app/)

---

## 🧠 About the Project

AI Smart Store Assistant is a smart shopping platform where users can:

* Browse products
* Get AI-based product recommendations
* Interact with a simple assistant to find products based on preferences

This project combines **e-commerce concepts + AI interaction + cloud deployment**.

---

## ✨ Features

* 🛒 Product listing interface
* 🤖 AI-powered recommendation endpoint
* ⚡ Fast backend using Node.js & Express
* 🌐 Cloud deployment using Google Cloud Run
* 📦 Containerized using Docker

---

## 🏗️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express
* **Cloud:** Google Cloud Run

## ⚙️ Installation & Setup (Local)

1. Clone the repository:

```
git clone https://github.com/your-username/ai-smart-store.git
cd ai-smart-store
```

2. Install dependencies:

```
npm install
```

3. Run the app:

```
npm start
```

4. Open browser:

```
http://localhost:8080
```
